Persebaran McDonald’s di Kota Bandung

Jelajahi lokasi setiap cabang McDonald’s di Bandung melalui visualisasi Peta Web GIS yang interaktif dan modern.

🍔 🍟

Tentang Proyek

Website ini merupakan implementasi Sistem Informasi Geografis (SIG) untuk memetakan data persebaran spasial.

Visualisasi Data Geospasial

Tujuan utama dari website ini adalah untuk menampilkan data persebaran cabang McDonald’s di Kota Bandung secara visual melalui sebuah peta interaktif. Proyek ini memanfaatkan data geospasial yang diolah menggunakan software QGIS.

Peta yang dihasilkan kemudian diekspor ke format web (HTML, JS, CSS) agar dapat disajikan dalam platform Web GIS yang modern, estetik, dan mudah diakses oleh siapa saja.

Peta Interaktif

Gunakan fitur pencarian atau filter berdasarkan kategori. Klik marker pada peta untuk melihat detail informasi setiap cabang.
